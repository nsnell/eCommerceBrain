The system

Two pieces of agent labor that should already be automated.

Almost every AI workflow for DTC dies in the same two places: maintaining context and maintaining tooling. We automated both.

01 / Context

It organizes its own brain. You just feed it.

Drop a customer survey, a campaign report, a competitor screenshot. Say "ingest this." The brain decomposes it into structured pages, updates the index, and links what's related.

Every skill loads only the pages it needs. An email skill pulls segments and brand voice. A landing page skill pulls competitors and positioning. No project file babysitting.

› ingest customer-survey-q1
02 / Skills

It writes its own skills from how you work.

Run captains-log at the end of a session. It records what skills ran, what worked, what felt manual, what needed rework. Quietly. One command.

Schedule level-up to run weekly. It reads every brief, spots patterns, and proposes new skills with draft instruction files. It builds the toolkit you would have built yourself, if you'd had the time.

› level-up weekly
The flywheel

Every session leaves the brain smarter than it found it.

Most AI workflows reset to zero. This one accumulates. Knowledge stacks, skills multiply, friction compounds out instead of in.

1

Ingest

Templates, surveys, sales calls, reviews. The agent decomposes raw input into structured pages.

2

Use

Skills read the brain first, then work. Output is grounded in your actual data, not best-practice clichés.

3

Log

Captains-log captures what ran, what worked, what stalled. One command at the end of work.

4

Evolve

Level-up reads briefs weekly, spots patterns, proposes new skills tailored to your workflow.

5

Compound

Brain gets richer. Toolkit gets bigger. Every future session starts further ahead.

Under the hood

No magic. Just markdown files and an agent that knows where to look.

Your brain is a folder of .md files you can read, edit, and version-control. The agent is the layer that keeps it organized and learns from how you use it.

What's inside

Four templates. Six skills. One brain that ties it together.

Pick the template you have the best data for and start there. You don't need all four before the brain becomes useful.

Templates

4 starters
icp-details.md Your customer's actual vocabulary, triggers, frustrations. Pulled from reviews, Reddit, support tickets.
brand-style.md How your brand sounds when content is working. Real patterns, not aspirational adjectives.
market-positioning.md What you own, what's contested, what you've ceded. Plus competitor snapshots.
customer-journey.md How people actually find you, what closes the sale, what makes them leave.

Skills

6 included
brain-ingest Turns raw input into structured pages. Templates, campaign reports, customer research, competitor intel.
brain-query Answers questions across multiple pages. Cites its sources.
brain-lint Health-checks the brain. Finds stale pages, contradictions, orphans, missing cross-references.
email-campaign-ideas Generates email concepts with angles, subjects, sequences, metrics. Grounded in brain knowledge.
captains-log End-of-session brief. Captures what ran, what worked, what felt manual.
level-up Reviews briefs weekly. Proposes new skills based on patterns. Checks for duplicates.
Honest framing

Who it's for, and who should skip it.

Built for you if

  • You run a DTC brand and already use Claude Code or Claude Cowork regularly
  • You've felt the friction of re-explaining your brand to every new chat
  • You have real customer data, reviews, or research collecting dust
  • You'd rather invest an hour upfront than re-prompt every session

Probably skip this if

  • You want a no-code chat UI with buttons. This is files, folders, an agent.
  • You haven't tried Claude Code or Claude Cowork. Start there first.
  • You want a magic content generator. It's only as good as what you feed it.
  • Your brand is brand new with no customer data to ingest yet.

Give your brand a memory.

Clone the repo, fill out one template, and run your first skill. The brain takes care of itself from there.
