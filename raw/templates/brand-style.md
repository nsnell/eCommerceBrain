# Brand Style

> **Load this file when the skill:** writes, rewrites, or edits any content in the brand's voice -- emails, product descriptions, social posts, landing pages, blog posts, ad copy, SMS, push notifications, or any output where tone and style matter.
>
> **Do NOT load this file when the skill:** only needs audience data, only needs competitive positioning, only needs the buyer's journey, or performs analysis/reporting tasks with no brand-voice output.

---

## How to Use This File

This file replaces a traditional "brand voice guidelines" PDF. Instead of aspirational adjectives like "warm, professional, and empowering," it captures the actual patterns from your best-performing content. Patterns a skill can match against, not vibes it has to interpret.

**To build yours:** Gather your 10 best-performing pieces of content (highest engagement, highest conversion, most shares). Feed them to Claude. Ask it to describe the voice, tone, and structural patterns it observes. Then validate: does the description match the winners but NOT your worst-performing content? That's the test.

## What to Do When You're Unsure

If a skill is generating content and isn't sure if it matches this style, apply this gut check:

1. Would the founder post this from their personal account without editing? If no, rewrite.
2. Read it out loud. Does it sound like a person talking, or a brand talking? It should sound like a person.
3. Could a competitor swap in their name and it'd still work? If yes, it's too generic. Make it specific.
